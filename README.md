# PicoRV32 — Minimal RISC-V CPU in Verilog HDL

## Description
A simplified PicoRV32 RISC-V CPU core built in Verilog HDL,
simulated using JDoodle (Icarus Verilog) online compiler.

## Project Details
- **Difficulty:** Intermediate
- **Duration:** 12–18 Days
- **Language:** Verilog HDL
- **Simulator:** JDoodle / Icarus Verilog

## Components Built
| Module | Filename | Description |
|---|---|---|
| Adder | adder.v | Basic 4-bit adder |
| ALU | alu.v | Arithmetic & Logic Unit |
| Register File | register_file.v | 32 registers storage |
| Decoder | decoder.v | Instruction decoder |
| Full CPU | picorv32_cpu.v | Complete CPU simulation |

## Learning Outcomes
- RISC-V architecture basics
- Parameterized RTL design
- CPU execution flow
- Verilog HDL programming
